# 🔐 Phishing Website Detection
### DSA-609 Mini Data Science Project

## 📋 Overview
Machine learning project to detect phishing websites 
using classification models.

## 📊 Dataset
- 10,000 website records
- 5,000 Phishing + 5,000 Legitimate (balanced)
- 48 features

## 🤖 Models Used
| Model | Accuracy | ROC-AUC |
|-------|----------|---------|
| Logistic Regression | 94.23% | 0.9834 |
| Random Forest | 98.57% | 0.9983 |

## 🛠️ Libraries
- pandas, numpy, matplotlib, seaborn
- scikit-learn

## 📁 Files
- `Phishing_Website_Detection.ipynb` - Main notebook
- `Phishing.csv` - Dataset
